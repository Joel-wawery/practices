# My Project: Practices

## Description
This is a practice project where I’m learning Node.js, TypeScript, Git, and general project workflow.  
It includes notes, experiments, and feature branches to understand version control and coding concepts.

## Tech Stack
- **Node.js** – for running JavaScript outside the browser  
- **TypeScript** – adds types to JavaScript for safer code  
- **Git & GitHub** – version control and code hosting  
- **VS Code** – development environment  

## Features / What’s Included
- Notes on JavaScript, Node.js, and TypeScript  
- Example of branching workflow with feature branches  
- Git commits representing daily “drafts” of learning
